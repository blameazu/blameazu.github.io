小意外
#

面試簡報繳交期限的當天，我跟 charhao 跑去吃晚餐

當牛排上桌的時候，我提了一下 : 「欸 阿你面試簡報有交嗎?」

charhao: 阿? 蛤? 要交?

我: 對阿，五點 (當下 4:30)

charhao: 幹 你在騙嗎?

我: (思考了一下)，喔對阿我在騙 哈哈

charhao: (看了一下手機)，喔幹，真的喔 靠

charhao: (起身衝回去學校拿他的筆電繳簡報)

只能說我真的是大好人，唉

面試當天早上
#

早上跟 charhao 早早搭一個 8:40 多的高鐵就跑上去了新竹

搭計程車跑去交大找 tw87

阿結果 tw87 還以為我們會去清大，所以他跑去清大找我們

還好青交有一個交清小徑(從清華走過來的話是 “青交"小徑)連通著

所以我們就邊聊天邊走著就找到 tw87 了

接著不知道要幹嘛 所以就跟 tw87 講一講幹話

然後我們就決定要先去吃點東西

想說去吃點新竹名產，結果走一走還是走到麥當勞…

進去賣噹噹跟 tw87 合買了一份 大薯 + 飲料的雙人餐，好吃

然後跟 tw87 下 chess，他依舊是讓我一個 boncloud，然後我依舊輸了 點點點

阿不知道為甚麼，吃完麥當勞之後突然決定要去吃新竹貢丸

所以我們就隨便走走

然後我剪刀石頭布輸了，所以跑去跟路人問路

然後路人是一個阿伯跟他兒子(?)，超級熱情，一直教我們怎麼走到對面的菜市場的貢丸店

阿結果我們最後隨便看了一家麵店裡面有賣貢丸，然後就進去吃了

結果三個人點三碗貢丸湯，剛開一把傳說對決

店長就端著一盤滷味放到我們桌上，說有一個男的已經付錢了，說要給 “三個帥哥”

阿我們當下很錯愕，而且我喝完貢丸湯就超級飽，所以就沒吃了

最後一盤的3/4滷味 都是 charhao 吃的，我跟 tw87 在旁邊講幹話，然後根本沒吃 超好笑

最後走的時候還是 charhao 跑去跟店長打包，然後我跟 tw87 放生他ww

走走走然後就回去清華了，原本是打算騎 ubike 回去交大宿舍

但是那邊都是山路，騎到腳快斷掉(超酸)

所以就還了 ubike (-10 元)

然後就打算用走的回去

結果 tw87 用他的神奇小手機租了一個一分鐘一塊的滑板車

然後就跟 charhao 一直練雙載(理論上是不行的ww)，一直練到摔倒，阿我一直在後面用跑的追 幹

阿路上有一件很好笑的事

charhao : 欸阿 你在交大有在社交嗎

tw87 : (不知道說了什麼)

(突然有一個女生騎著滑板車從山坡上下來，然後跌倒了)

charhao : 欸 趕快去社交，就跟他說 “同學，我覺得你跌倒的姿勢特別可愛，可以加一下你的 IG 嗎”

超好笑，超壞

然後我們就跟著 tw87 回去他的宿舍，跑去他的地下室做壞壞的事情

像是很大聲的玩籃球機，很大聲的玩桌上一個盤子在碰碰然後要掉進別人的洞裡的遊戲(不知道那個叫什麼)

反正就是很大聲 哈哈

最後就跑回去清大準備面試了

面試準備期間
#

面試地點在清大六樓

一開始進去報名，然後就跑去休息室

一進去就看到 temmie 在發特上檸檬紅茶，只是 tw87 說喝特上的都不會特上，所以我就沒拿了

然後跑去另外一間教室聽大家講廢話，有看到 winlu、boss_zz、yok、koying、max甚麼的(記不起來id)

阿 tw87 一直在講屁話，所以就輕鬆的講講講，然後就要面試了

不過 tw87 在面試前有講一題史萊姆的題目，然後這時候 msqrt 有來，但是我當下忘記他是誰了ww(只覺得很眼熟)

然後在面試前 2 分鐘我還發現我的名牌上的准考證號碼是錯的 幹

然後就被帶去面試了

面試
#

第一間教室
#

第一間教室有兩個教授

還有投影幕

我一進去就是講簡報內容

但我忘記要先自己模擬一下了，所以整個講超爛

反正就是要切下一幕我才知道我簡報在寫三小

超混亂，而且我整個人都超抖、超緊張

然後講完後，教授就問我幾個問題

教授: 你今年有進全國賽嗎?

我: 有

教授: 有補件嗎?

我: 有

教授: 關於你 TOI 初選第三十名，好像表現的不是像其他比賽一樣的亮眼，是怎麼了嗎?

我: (唬爛一下，反正就說我最後選題失誤了)

教授: 你去年全國賽沒拿獎，是發生什麼事了嗎

我: 精神狀態不佳，(然後講了一大堆東西，最後說今年目標二等獎)

教授: 應該要把目標放在一等獎(笑)

我: 哈哈(哈哈 超好笑)

(一直沒有講話的教授) : 阿你有要繼續參加選訓嗎

我: (沉默了一下，困惑了一下) 痾，我會

教授: 你們學校有來面試的除了你還有另一個人?

我: 對，我跟他很熟，常常一起討論問題

教授: 你們有去比 YTP?

我: 對，跟另外一個組隊的

教授: 不是三個人嗎

我: 還有一個學弟跟我們組隊

教授: 那你們專題做甚麼?

我: 痾，其他人要讀學測，所以我們沒有做專題

教授: (笑)

教授: 你們學校校隊是有多少個人比，取多少個

我: 初選大概 30~40 個人比，取 10 個，複選取 5 個

教授: 那你覺得你們學校的校隊在區賽的獲獎機會?

我: 校隊的人都很強，應該都可以拿獎

然後就結束了

反正我覺得教授應該是想不到甚麼東西問，所以就隨便亂聊

然後第一個教授全程一直對著我點頭

然後第二個教授全程安靜，問了一個三小問題就安靜的聽

整體感覺有點輕鬆

第二間教室
#

第二間教授也有兩個教授

然後桌子上有很多個信封，每個信封上都有數字

教授: 可以自己抽一個信封，然後有需要也可以使用白板

我: 好(抽了 5 號袋)

看了一下，好像是象棋的棋盤，馬在上面走，要怎麼知道與每個格子的最短距離、以及最短距離下的方法數

我: 好了，首先我會採用 BFS 的算法

教授: 蛤，可以講大聲一點嗎，我耳朵有點問題

我: 我會採用 BFS 的算法

教授: 複雜度呢?

我: $O(NM)$

教授: 阿方法數你要怎麼計算

我: 阿? 我沒看到，等我一下

我: 我會採用排列組合的加成性(應該是加法原理，根本不知道我在講什麼)

教授: 阿? 什麼?

我: 就是動態規劃然後紀錄每個格子的方法數，然後到那個點那個點，然後就把來的地方加起來 (在空氣中比劃)，我最後應該也會採取 BFS 的方法

教授: 複雜度呢?

我: 應該一樣是 $O(NM)$

教授: 好你可以抽下一題了

我抽了第六題

看了一下，一張圖數三環，然後假如邊數很大我要怎麼做，邊數很小我要怎麼做

我: 我覺得我會先採用 bitset 的方式，就枚舉兩個點，然後把他們壓起來找

教授: 複雜度呢?

我: $O(N^2)$

教授: 恩? 是嗎?

我: 喔不對因為我用到 bitset，所以應該是 $O(\frac{N^3}{d})$

教授: 好，那複雜度可以更好嗎

我: 恩，我有可能會採用 DFS 的方式，就走下去然後找回邊

教授: 阿? 那你要怎麼數到包住的三環個數?

我: (想了一下)喔 我好像想錯了，那我應該想不出來了

教授: (笑了一下) 我覺得你可以從枚舉邊開始想

我: 喔對，那我可以枚舉邊的兩點然後檢查

教授: 複雜度呢?

我: $((MN) \log N)$，假如有用 set 找邊存不存在的話

教授: 能更好嗎?

我: (不知道講了什麼)

教授: 好像可以 $O(M^2)$

我: 歐對對對，應該就那樣那樣，這樣這樣 (根本不知道怎麼做)

教授: 好

我: 那我需要再抽下一題嗎(想說時間還夠，我應該要盡力表現自己)

教授: 我覺得你表現的很好了，不用再講題目了

我: ….(?)

教授: 喔對了 順帶一題，你剛剛那題其實可以做到 $O(M \sqrt{M})$

我: 喔? 酷喔

教授: 你可以回家了解一下

我: 好

教授: 你就按照度數去排，然後前面指向後面，枚舉下一個點的下一個點，然後檢查

教授: 以分析的角度這樣會是根號

我: 好

教授: 好 那今天應該到這邊，謝謝你

我: 謝謝教授

我: 我可以問一個問題嗎?

教授: 可以

我: 我剛剛講的都是對的嗎?

教授: 痾對，你講得很好

教授: 恩? 是哪一部分…?

我: 我剛剛講的解都是很快想出來的東西

我: 我正式賽應該都是先將寫法寫出來然後再去證明他的正確性

我: 所以不太敢保證一瞬間想出來的想法是對的

(兩個教授): 喔不會不會，你講的很棒，對於面試這幾分鐘來說，能瞬間想到一個想法就很不錯了

我: 喔好 謝謝教授

然後就敲門了(8 分鐘到)

我就走出去了(感覺是被趕出來的ww)

面試後
#

跟 charhao 還有 tw87 跑去逛大學書店 (聽說沒有大學的不能進去)

結果 tw87 手握文化幣 1200 巨款，結果沒有買任何東西(包括寫真集)

超爛

然後晚餐本來要去吃肥仔龍(不知道是不是這樣念)

結果 tw87 自己怕了，所以我們隨便找一間牛排館進去吃了

最後又跑回去交大打球完，跑去圖書館(或者是某種讀書的地方) 喝飲料講幹話 + 打牌

最後下去等公車順便聽 tw87 說 實況不死鳥的解法，然後就回家了 omob
