題目連結
#

ABC 150 E

題目
#

ABC150 E - Change a Little Bit

今天有兩個長度為 $N$ 的 01 序列 $S$ 跟 $T$

定義一次操作為

選擇 $S$ 的一個位置 $i$

花費 $C_i \times D$，$D$ 是 $(S_j \neq T_j, \forall 1 \le j \le N)$ 的個數，翻轉 $S_i$ ($0 \to 1, 1 \to 0$)

定義函數 $f(S, T)$ 代表著序列 $S$ 經過一系列操作後等於序列 $T$ 的最小花費是多少。

總共會有 $2^{N} \times (2^{N}-1)$ 的 $f(S, T)$ 組合，計算 $\sum f(S, T) \pmod {10^9+7}$，也就是所有情況的最小花費總和模上 $10^9+7$

$1 \le N \le 2 \times 10^5$

$1 \le C_i \le 10^9$

題解
#

弱化問題
#

一開始先要想最小花費的策略

所以先把題目弱化為

給定 $S$ $T$ 求最小花費

Greedy
#

我們發現我們對於按鈕的順序性根本用不到

所以我們考慮 sort 整個陣列

使得 $C_i$ 是遞增的

那麼我們可以輕鬆的想到由小到大的翻轉 $S_i \neq T_i$ 會是最佳的

證明
#

Proof

對於兩個 $S_i \neq T_i$ 不相同的位置 $j, k(j < k \Rightarrow C_j \le C_k)$ 來講

我們依照剛剛的假設會得出

$(\text{先選 } j \text{ 再選 } k) \le (\text{先選 } k \text{ 再選 } j)$

也就是

假如目前有 $D$ 個不一樣的位置

那麼我們可以得到以下式子

$C_j \times D + C_k \times (D-1) \le C_k \times D + C_j \times (D-1)$

左右共同減去 $(C_j + C_k) \times (D-1)$

$\Rightarrow C_j \le C_k$

選擇出發點
#

對於原本的問題，也就是所有種情況要怎麼做呢?

因為有 $4^N$ 量集種可能

所以我們可以考慮一下排列組合

拆貢獻
#

對於位置 $i$

我們其實可以將它對於答案總和的貢獻拆出來

也就是我們要計算

位置 $i$ 對於總體答案(所有情況)的總共貢獻

先想想我們要怎麼找
#

對於狀態 $S$，總共有 $2^N$ 種情況

那麼我們只要決定好 $S_j, T_j, \forall 1 \le j \le N$ 是否相等

那麼我們就可以對應出底下的 $T$

所以我們可以想像我們已經有 $2^N$ 個 $S$ 了

對於每個 $S$ 狀態，我們分配不相同的位置給他，那麼自然就對應出底下的 $T$ 的狀態了

分析
#

對於狀態 $S$，以及考慮翻轉 $i$

我們會發現我們根本不在一 $1 \sim i-1$ $T$ 的狀況

因為到目前狀態 $S$ 考慮翻轉 $i$ 的情況下

我們肯定已經將 $1 \sim i-1$ 都翻轉到相同了

所以左邊藍色的情況總和就是 $2^{i-1}$

對於紅色($i$) 來說，既然我們要翻轉 $i$，則 $S_i \neq T_i$

剩下右邊綠色的還沒考慮

枚舉什麼
#

我們枚舉 $D$，代表著對於 $i+1 \sim N$ 有 $D$ 個 $S_j \neq T_j$ 的位置

那麼我們本次翻轉的費用就是 $C_i \times (D + 1)$，加一是因為還要算到 $i$

對於每個 $D$ 來講，會有 $\binom{N-i}{D}$ 個可能

因為要在綠色的框框裡面 $(N-i)$ 挑 $D$ 個東西

那式子是不是就變成了

$$ \sum^{N}_{i=1} 2^N \times 2^{i-1} \times \sum^{N-i}_{D=0} (\binom{N-i}{D} \times C_i \times (D+1)) $$

優化
#

上面的式子好像還不夠優秀

因為要算的複雜度高達 $O(N^2)$

所以我們再整理一下式子

$$ 2^N \times \sum^{N}_{i=1} 2^{i-1} \times C_i \times \sum^{N-i}_{D=0} (\binom{N-i}{D} \times (D+1)) $$

發現我們的瓶頸是卡在

$$ \sum^{N-i}_{D=0} (\binom{N-i}{D} \times (D+1)) $$

那我們要怎麼算呢?

重頭戲 — 生成函數
#

以下看不懂的話，那就假裝你看得懂就行

設一多項式 $f(x)$ 為 $(1 + x)^{N-i}$

那麼要求 $\sum^{N-i}_{D=0} \binom{N-i}{D}$

就是 $f(1) = 2^{N-i}$

原理是二項式展開 $f(x)$

$f(x) = \binom{N-i}{0} x^0 + \binom{N-i}{1} x^1 + \binom{N-i}{2} x^2 + \ldots + \binom{N-i}{N-i} x^{N-i} = \sum^{N-i}_{D = 0} \binom{N-i}{D} x^D$

再來通靈一下

假如有一個多項是 $g(x) = \sum^{N-i}_{D = 0} (\binom{N-i}{D} \times (D+1) \times x^D)$

那麼 $g(1)$ 會是我們要的答案

那麼 $f(x)$ 跟 $g(x)$ 差在哪裡呢?

沒錯! 差在多出來了一個 $(D+1)$

那我們仔細觀察一下 $f(x)$

看哪裡有 $(D+1)$ 的痕跡

$$ f(x) = \sum^{N-i}_{D = 0} \binom{N-i}{D} x^D $$

微分碎蓋
#

欸? $D$ 在 $x$ 的次方上欸

所以我只要對於 $f(x)$ 做一次微分，是不是就可以把 $D$ 拿下來了!

$$ f'(x) = \sum^{N-i}_{D = 0} \binom{N-i}{D} \times D \times x^{D-1} $$

蛤，但我要的是 $D+1$ 又不是 $D$!

升階
#

我們直接將 $x$ 的次方數硬頂個 $+1$ 上去不就好了?

$$ x \cdot f(x) = \sum^{N-i}_{D = 0} \binom{N-i}{D} x^{D+1} $$

再對他做微分

$$ (x \cdot f(x))' = \sum^{N-i}_{D = 0} \binom{N-i}{D} \times (D+1) \times x^{D} $$

蛤那我有化簡到嗎?
#

我們回溯回去好久好久以前

$f(x)$ 長怎樣?

$$ f(x) = (1 + x)^{N-i} $$

那根據剛剛的操作又會變怎樣?

$$ x \cdot f(x) = (1 + x)^{N-i} \cdot x $$$$ (x \cdot f(x))' = ((1 + x)^{N-i})' \times x + (1 + x)^{N-i} \times (x)' $$$$ (x \cdot f(x))' = (N-i) \times (1 + x)^{N-i-1} \times x + (1 + x)^{N-i} \times (1) = g(x) $$

所以 $x$ 代 $1$

得到

$$ g(1) = (N-i) \times 2^{N-i-1} + 2^{N-i} $$$$ g(1) = (N-i) \times 2^{N-i-1} + (2) \times 2^{N-i-1} $$$$ g(1) = (N-i + 2) \times 2^{N-i-1} $$

代回原式
#

經過剛剛的一番激戰

我們將答案算的差不多了

我們知道了

$$ \sum^{N-i}_{D=0} (\binom{N-i}{D} \times (D+1)) $$

就等於

$$ (N-i + 2) \times 2^{N-i-1} $$

那麼原式會是長

$$ 2^N \times \sum^{N}_{i=1} 2^{i-1} \times C_i \times (N-i + 2) \times 2^{N-i-1} $$

整理一下

$$ 2^N \times \sum^{N}_{i=1} 2^{N-2} \times C_i \times (N-i + 2) $$$$ 2^{2N - 2} \times \sum^{N}_{i=1} C_i \times (N-i + 2) $$$$ 4^{N - 1} \times \sum^{N}_{i=1} C_i \times (N-i + 2) $$

耶 $O(N)$ 就可以算完了

實作
#

Spoiler 
const int mod = 1e9+7;
int n; cin >> n;
int ans = 0; // 答案
vector<int> C(n+1);
for(int i = 1; i <= n; i++) cin >> C[i];
sort(C.begin()+1, C.end()); // 由小到大排序

// 計算答案，也就是 \sum^{N}_{i=1} C_i \times (N-i + 2)
for(int i = 1; i <= n; i++) {
    ans = (ans + 1ll*C[i] * (n-i+2))%mod;
}
// 最後要乘上 4^{N-1}
for(int i = 1; i < N; i++) ans = 1ll*ans * 4 %mod;
cout << ans << '\n';

做完了，耶
