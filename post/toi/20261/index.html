Day 1
#

前言
#

三年只進過這次選訓

然後三年入營考都沒考進選訓過

高三這次是因為全國賽打的比較好，所以就用二等獎混進來了

正題
#

公假
#

在 TOI 前一個禮拜就請公假

但大概都是早上起床很不想去學校 -> 到學校吃早餐玩傳說

-> 打 atcoder 題目 -> 想不出來 -> 擺爛

段考
#

因為選訓結束後的後一周是段考

因為沒辦法準備段考(雖然我也不會準備 ㄏㄏ)

所以可以按照學校制度去申請免試

但要跟各科老師協調

國文老師說沒有段考，但學習單要全交…

英文老師說建議我要考段考，不然我會被當掉

物理老師說不用考段考

化學老師問我說我學分還夠嗎，意思差不多就是我已經被當掉了

幹

準備
#

把幾件衣服跟口罩塞進行李箱

然後把筆電跟綠皮書還有英文雜誌放到書包裡面

就準備完了

不過有先準備好入營保證書 & 切結書 & 身分證 & 學生證

不然就跟南一中的學弟們一樣進不去選訓營了

早上
#

早上很早就起床了，大概 5.30

然後準備好行李就出發高鐵站

搭早上 7:07 台南到台北的車

特別早

然後因為是星期一，所以我跟 brinton 天真的以為自由座會很少人

買了自由座

然後被罰站一小時半 幹

卡恩爆揍時間
#

到台北後就先搭捷運到公館

然後把很大很大的行李箱丟進寄放櫃裡面

接著跑去德田館聽台大的課

早上 9:10 的課，卡恩教圖論

那些圖論大概都是論文等級的

像甚麼 endos … 之類的看起來連英文都不是的演算法名稱

就特別種

然後就一路聽一大堆看不懂的名詞還有數學證明

然後就昏昏沉沉的結束了

一想到台大要上這些課，就特別壓力

這堂課上也看到一大堆台大的大佬

中午
#

中午時間就從台大走回去師大會館

然後因為聽大家講師大會館很破

所以 brinton 看到師大本部後面有一個綠色鐵皮屋

然後就很斷定後面的鐵皮屋一定是師大會館

結果才發現前面的一棟 很高很大 很像圖書館的建築物才是師大會館

感覺也沒到多差

喔然後順便把行李寄放在師大會館

下午
#

因為是下午 4.30 報到

聽 Brinton 說只要是國手就要爬象山

所以我跟 brinton 就決定跑去爬象山

不過在搭紅線到象山站之前

我們先跑去 101 吃中餐

吃飽了才跟 brinton 搭捷運跑去爬象山

一開始爬到象山的半山腰之前都很快

我就一直衝衝衝

因為一停下來感覺腳很酸，所以想要很快爬上去再下來

結果到一半的時候就整個都很喘，差點趴在地板上抽

只好休息了好幾下

然後再一路辛苦的爬到最頂端

然後拍了幾張照，就準備下山

阿 brinton 說下山走原路就真的很遜

所以我們就越過象山，往一個不知道叫什麼的平台走

然後就緩坡 + 陡坡的一直下下下去

不過在下到一半的時候有遇到一個印度人

他用很像 YT 上的印度英文跟我們講話

然後 brinton 很會社交

幫他拍照，雖然他拍的照片都是糊的

最後讓印度人拿出看起來很專業的專業相機讓 Brinton 拍

然後就握個手然後就下山了

不過我們可以總結這段旅程

在深山老林中遇見印度資訊大佬，教會我們怎麼打爆大家 AK IOI

報到
#

爬完象山後大概 3. 多

所以我們就先跑回師大會館拿選訓的同意書

然後就騎腳踏車跑去師大分部(因為要在那裏報到)

報到之後就進去教室

看到好多大佬

leolinorz
#

接著就聽選訓的人講規則跟一些規則

順便知道一階住師大會館 二階住捷思旅是因為某些學長

然後還被一公尺大的披薩騙到了(根本沒有)

晚餐就吃披薩

意外的發現蔡孟平根本是晚餐大師

一直在吃東西

然後吃完晚餐後就回去教室聽 leolin 嘴砲別人

順便聽了一些怪話，像

Brinton : 我也會唬爛阿，你要去月球的話，你就每天一直跳，跳著跳著就跳上去了

超怪

回師大會館
#

屁話時間完後就搭遊覽車回師大會館

順便發房間

我是跟兩個很強的人住的

一個有進過選訓 一個國三

反正就超級電

然後我們住七樓，我跟蔡孟平扛著行李廂從一樓上去

真的很怪

然後師大會館的熱水器真的很怪，忽冷忽熱 嚇到我了

廢話
#

第一天感覺選訓好像很輕鬆愉快

然後大家都好強喔

然後第一天沒拿到 introduction to algorithm

不知道是沒有要發還是之後才會發，反正好傷心

然後晚上有 CF 大家好像要打

但我感覺好無聊，不知道要不要先睡

先小小期許自己可以把模考打好

然後住到捷思旅 順便翹掉第二次段考 ㄏㄏ

後記
#

CF
#

晚上跑去交誼廳跟大家打 CF

但我就覺得我很爛，所以報 rated 打算寫個 pA 來掉分

但不知道為什麼就跟蔡孟平跑去戳 pG

然後手算 sample 1 就怎麼算都算不出來

過了很久突然靈光一現，懂題目再講甚麼後

又被蔡孟平的變數限制觀察法發現是快速冪

然後我就很唬爛的唬爛出一個 dp 狀態跟轉移矩陣

然後就開開心心的吃了 TLE

但因為黃宇綸特別種，他會把矩陣乘法壓 8 倍常數

所以黃宇綸施展他的絕活後

我們就過了 pG ㄏㄏ

Day 2
#

早上
#

師大會館的冰箱
#

很吵…

他會一段時間循環然後發出怪怪的聲音

有點像有人在撬你的門鎖的感覺

晚上房間是黑黑的，聽起來有點可怕

起床
#

大概 6. 多就自己醒來了

睡眠品質沒多好

然後下到大廳聽到大家在抱怨師大會館的床太短

我感覺沒甚麼感覺

有可能是我沒有住過捷思旅，也有可能是我太矮了

傳奇繞路王
#

搭遊覽車去師大

司機看我們太想睡覺了

特別會繞路

大概開了十五分鐘還在台電大樓站

特別強

所以我們就有點被小 delay 到了

這裡很重要，司機之後會再被提到

早餐
#

早餐是放在教室

好像是吐司夾肉 + 紅茶

還不錯吃

上課
#

好像是上數學

但我太矮了，前面的人都太高

直接把我的視線擋住，想要看剪報還要左躲右躲

再加上他在講一些看起來好不重要的東西

所以我就不爭氣的睡著了

vir
#

睡醒後發現 brinton 他們不揪我就跑去打 KOI

我特別傷心

所以就跑去找唐狗針還有林宥辰打 atcoder

然後林宥辰還特別跟我講他那場有打過但 PC 燒雞

所以等我秒掉 pA pB 後

我就燒雞 pC 了…

燒了差不多三十分鐘後才發現題目限制其實很小

所以我根本不用怪怪的數學去算

就真的很燒雞

然後暴力幹一幹就算出來了

接著把 pD 做掉

pE 好像是 floyd warshell 變種 + 暴力擴邊

但唐狗針特別強，他發現我努力在維護還多吃一次 WA 的東西根本用暴力算就行

pF 被蔡孟平的狀態設計種到了

我樹 DP 用了一個拉完了的狀態去跑

難怪超難刻出來

但我現在還是不會 pF 燒雞

中午
#

午餐
#

午餐是吃師大的自助餐

然後 TOI 是提供每個人 120 的扣打去吃

阿我不知道要吃甚麼 所以跟著 boss_zz 吃

但他挑茄子，但我不吃，特別哎呀呀

然後我總共花 73 元一盤，感覺太省了

雖然吃的也算飽

但就感覺虧了

不過還是見證了吃飯國手 蔡孟平 的威力

他隨便夾一夾就 160 不愧是蔡孟平 orz

下午
#

刷題
#

下午是試題練習時間

就是 TOI 有給一個 cms 系統

然後有很多題

題目都是 TOI 新手同好會的題目

阿 TGOI 是有計分，TOI 沒有

我覺得好無聊，再加上想刷點水題

所以就揪著唐狗針一起來寫

然後就一直刷刷刷

cms WTF
#

其中有一題特別強

他的題目名稱是 power

然後我用一個 $O(10^8)$ 的解傳上去

結果 TLE

而且這個 $10^8$ 常數還很小

但就是 TLE

不太理解

刷完了
#

刷著刷著就刷完了

整個下午的心得就是唐狗針特別會想實作細節

沈宗瑞特別會嗆我經典題多帶兩個 log

林宥辰是數學 & 圓切割之神

吃晚餐
#

晚餐吃一個小型 buffet

不在師大學餐裡面吃了

搬到教室外面的教室裡面吃

感覺就普普的

然後聽大家講怪話就吃完了

打球
#

吃完晚餐離 becaido 的課還有一段時間

所以我們就一群人跑去籃球場打球

發現魚切特別會打籃球，根本魚切絕活

然後蔡孟平腳趾受傷也可以打超好

結論根本是台北人特別會打球，卷

不過很好笑的是比賽的裁判是夜未央

他應該是年齡差太多，我有點跟不上他的邏輯

反正就很強

不過打球的時候 brinton 跟另一個物奧神有點小火花

整個場面有點尷尬的結束了

becaidorz
#

回去上課的時候發現 caido 也一起來了

然後真的覺得業為央長的超像 caido 超好笑

becaido 跟 PCC 來講 TOI 初選的題解

但被抓到我完全不會 TOI 初選

所以根本聽不懂題解

不過我有蠻認真聽他們的模考策略

PCC 算是特別會一輪一輪刮分

becaido 算是特別會一輪一輪的破台

聽起來就是要當國手就要很強

跑步
#

結束後要回師大會館

輔導員跟我們說要先去跑操場一圈

我都還以為是嘴砲而以，結果真的要跑

燒雞

傳奇司機
#

跑完操場後就集合去等車回師大會館

結果等了快一個小時的車

誕生了傳奇怪話

輔導員 : 等一下司機要開過來了，他有點不爽，所以你們安靜一點

我們 : ?

反正就很怪

師大夜市
#

回到師大會館後

我們就又一群人跑出去師大夜市買東西吃

我買一杯薄荷檸檬，感覺又酸又爽 蠻好喝的

而且還可以喝到檸檬果粒 讚

BOTC
#

回到師大會館後，查完房

大家就跑去交誼廳玩 BOTC

大概就是變種狼人殺

很種的那種

反正我身分是完全沒有記住

大家都邊玩邊查

但大家大概都是第一次玩

就整個場面就超怪

私聊變成 8 人公聊

然後好人身分全都跳光光，壞人被誤打誤撞的打爆

然後就結束了

總之就是蠻好玩的，雖然沒懂大家在講甚麼

反正第一輪把 brinton 是處女查出來就特別強大 哈哈

btw 大家的聲音真的有點大聲

感覺明天會被物奧的人投訴

Day 4
#

前言
#

先說一下，我在選訓的時候只有打前三天的文

然後之後大概都在跟大家玩 BOTC 或者睡覺

所以沒有打心得

接下來的都是回憶中的心得

所以也許不會那麼詳細

早上
#

早上是韓永楷的課

他就是那個清大特選面試聽不出來 $n$ $m$ 念起來差在哪裡的教授

反正他教了一些很水的東西

差不多 30min 就說完了

爽爽領 9000$ 就跑路了

對他印象沒那麼好

下午
#

下午好像跟大家去打球之類的 有點忘記了

或者玩 BOTC

Day 5
#

早上
#

早上是尉崴跟廖恆的經驗分享

聽廖恆講心得的時候有點羨慕

羨慕廖恆這麼現充又可以把比賽打好

然後學習速度很快，根本人生勝利

聽尉崴講的時候覺得進二階好困難

一模贏快 100 多分還被逆掉，聽起來就超可怕

然後聽國培的時候差點睡著

因為是一些旅遊有關的東西

聽起來沒甚麼興趣

不過還是有聽到一些些 brinton 玻利維亞拉屎的光榮事蹟 ㄏㄏ

下午
#

下午是黃光瑞的課

前面聽了一些 基礎 STL 的運作原理

才發現 heap 原來是可以用乘法除法做到實作在一個陣列上的

大受震撼

然後也發現 稀疏的 list 根本就是 vector

完全被發現根本不會基礎 C++

然後也有聽一些 interval tree，但感覺沒那麼強，所以沒認真聽

吃完飯
#

吃完飯後跟大家打種馬盃

第一題我 rnk.2

第二題我 rnk.10

第三題以後的題目我都寫不出來了，全都最後一名

根本被 CF 單防，害我整個晚上超不爽

然後 babrian 一直說他打很爛，然後嗆我有打跟沒打一樣

我差點破防

晚上的課
#
