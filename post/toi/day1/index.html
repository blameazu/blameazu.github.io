前言
#

三年只進過這次選訓

然後三年入營考都沒考進選訓過

高三這次是因為全國賽打的比較好，所以就用二等獎混進來了

正題
#

公假
#

在 TOI 前一個禮拜就請公假

但大概都是早上起床很不想去學校 -> 到學校吃早餐玩傳說

-> 打 atcoder 題目 -> 想不出來 -> 擺爛

段考
#

因為選訓結束後的後一周是段考

因為沒辦法準備段考(雖然我也不會準備 ㄏㄏ)

所以可以按照學校制度去申請免試

但要跟各科老師協調

國文老師說沒有段考，但學習單要全交…

英文老師說建議我要考段考，不然我會被當掉

物理老師說不用考段考

化學老師問我說我學分還夠嗎，意思差不多就是我已經被當掉了

幹

準備
#

把幾件衣服跟口罩塞進行李箱

然後把筆電跟綠皮書還有英文雜誌放到書包裡面

就準備完了

不過有先準備好入營保證書 & 切結書 & 身分證 & 學生證

不然就跟南一中的學弟們一樣進不去選訓營了

早上
#

早上很早就起床了，大概 5.30

然後準備好行李就出發高鐵站

搭早上 7:07 台南到台北的車

特別早

然後因為是星期一，所以我跟 brinton 天真的以為自由座會很少人

買了自由座

然後被罰站一小時半 幹

卡恩爆揍時間
#

到台北後就先搭捷運到公館

然後把很大很大的行李箱丟進寄放櫃裡面

接著跑去德田館聽台大的課

早上 9:10 的課，卡恩教圖論

那些圖論大概都是論文等級的

像甚麼 endos … 之類的看起來連英文都不是的演算法名稱

就特別種

然後就一路聽一大堆看不懂的名詞還有數學證明

然後就昏昏沉沉的結束了

一想到台大要上這些課，就特別壓力

這堂課上也看到一大堆台大的大佬

中午
#

中午時間就從台大走回去師大會館

然後因為聽大家講師大會館很破

所以 brinton 看到師大本部後面有一個綠色鐵皮屋

然後就很斷定後面的鐵皮屋一定是師大會館

結果才發現前面的一棟 很高很大 很像圖書館的建築物才是師大會館

感覺也沒到多差

喔然後順便把行李寄放在師大會館

下午
#

因為是下午 4.30 報到

聽 Brinton 說只要是國手就要爬象山

所以我跟 brinton 就決定跑去爬象山

不過在搭紅線到象山站之前

我們先跑去 101 吃中餐

吃飽了才跟 brinton 搭捷運跑去爬象山

一開始爬到象山的半山腰之前都很快

我就一直衝衝衝

因為一停下來感覺腳很酸，所以想要很快爬上去再下來

結果到一半的時候就整個都很喘，差點趴在地板上抽

只好休息了好幾下

然後再一路辛苦的爬到最頂端

然後拍了幾張照，就準備下山

阿 brinton 說下山走原路就真的很遜

所以我們就越過象山，往一個不知道叫什麼的平台走

然後就緩坡 + 陡坡的一直下下下去

不過在下到一半的時候有遇到一個印度人

他用很像 YT 上的印度英文跟我們講話

然後 brinton 很會社交

幫他拍照，雖然他拍的照片都是糊的

最後讓印度人拿出看起來很專業的專業相機讓 Brinton 拍

然後就握個手然後就下山了

不過我們可以總結這段旅程

在深山老林中遇見印度資訊大佬，教會我們怎麼打爆大家 AK IOI

報到
#

爬完象山後大概 3. 多

所以我們就先跑回師大會館拿選訓的同意書

然後就騎腳踏車跑去師大分部(因為要在那裏報到)

報到之後就進去教室

看到好多大佬

leolinorz
#

接著就聽選訓的人講規則跟一些規則

順便知道一階住師大會館 二階住捷思旅是因為某些學長

然後還被一公尺大的披薩騙到了(根本沒有)

晚餐就吃披薩

意外的發現蔡孟平根本是晚餐大師

一直在吃東西

然後吃完晚餐後就回去教室聽 leolin 嘴砲別人

順便聽了一些怪話，像

Brinton : 我也會唬爛阿，你要去月球的話，你就每天一直跳，跳著跳著就跳上去了

超怪

回師大會館
#

屁話時間完後就搭遊覽車回師大會館

順便發房間

我是跟兩個很強的人住的

一個有進過選訓 一個國三

反正就超級電

然後我們住七樓，我跟蔡孟平扛著行李廂從一樓上去

真的很怪

然後師大會館的熱水器真的很怪，忽冷忽熱 嚇到我了

廢話
#

第一天感覺選訓好像很輕鬆愉快

然後大家都好強喔

然後第一天沒拿到 introduction to algorithm

不知道是沒有要發還是之後才會發，反正好傷心

然後晚上有 CF 大家好像要打

但我感覺好無聊，不知道要不要先睡

先小小期許自己可以把模考打好

然後住到捷思旅 順便翹掉第二次段考 ㄏㄏ

後記
#

CF
#

晚上跑去交誼廳跟大家打 CF

但我就覺得我很爛，所以報 rated 打算寫個 pA 來掉分

但不知道為什麼就跟蔡孟平跑去戳 pG

然後手算 sample 1 就怎麼算都算不出來

過了很久突然靈光一現，懂題目再講甚麼後

又被蔡孟平的變數限制觀察法發現是快速冪

然後我就很唬爛的唬爛出一個 dp 狀態跟轉移矩陣

然後就開開心心的吃了 TLE

但因為黃宇綸特別種，他會把矩陣乘法壓 8 倍常數

所以黃宇綸施展他的絕活後

我們就過了 pG ㄏㄏ
