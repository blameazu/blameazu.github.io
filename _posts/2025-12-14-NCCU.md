### 測機

開幕拖了點時間，然後就上去三樓測機了

緊張的打開題本，發現題目跟去年測機一模一樣 超爛

然後我就仔細的看了一下 pA、pB

pA 稍微刻一下幾何模板就過了

pB 雖然知道是 merge sort，但是因為是 2 step，所以不太知道怎麼刻函式

本來想去找 attachment 的 grader.cpp 跟 sample code

結果沒找到

然後我看 communication 也沒有人提問

所以我還以為是只有我沒找到而已

然後我就很認真找了一下

usb、網站都翻過了 沒有

然後就認命的馬農一下把 code 打出來

傳上去吃挖然後調 code

然後吃挖，直到announcement 說 pakage 有放上去

然後我載下來發現只有 sample code，草

所以我只能對著我的 code 發呆，發動究極瞪眼法，然後就 debug 出來了

傳上去就 ac 了

接下來好像剩 15 min

我不看 pC 就知道是數獨了

然後本來想去載 input

結果翻遍了也是沒找到

所以覺得也是主辦方在搞，所以我就通靈一下直接在 vscode 打一份隨機解的 code

打好 debug 完後 主辦方也放上去了

input 載下來跑一下，剩下 $1 \sim 2$ 分鐘，所以我就跑跑跑，直接傳

結果 0 分，我嚴重懷疑不是我 code 寫錯，那就是他根本沒設 grader or checker

雖然前者很大機率，但第二、三題出包害我有點不太信任他們

## 全國賽

### 早上

早上起床吃個 buffet，然後發現變的超級破

沒有薯條也沒有雞塊

雖然有去年的土司，但是感覺不好吃

所以我吃了烤土司跟蔥抓餅跟蛋花

然後吃了點水果，本來想喝點咖啡，不過還是算了

然後退房，上遊覽車，回去政大

### 打比賽

到政大後，就被趕上樓去打比賽

不過政大的工作人員有提醒我們要拿學生證在身旁(然後兩天根本沒用到，真的不知道在幹嘛)

然後我也有把貼紙貼在我的御茶園大罐罐上，旁邊放我的鉛筆盒跟吸吸吸果凍

接著就滾進去考場考試了

### 正式賽

**以下有大量暴雷 嘻嘻**

**策略**

原本我以為這次題目會跟上次全國的難度差不多，所以給自己的目標是每一題大概花 30 min

**解題過程**

一開始我開 pA，發現很水，所以就解掉了

那時候好像 10 min 之類的，發現我第二題可以想 30 - 10 + 30 min，特別爽

所以跑去看題目，然後看了 10 min 還是不知道題目在講什麼

生氣，所以出去吃吸吸吸果凍，喝茶

然後回來看 pC

發現 pC 好像很好做，然後 bellman ford 可以做完 1、3 子題

所以馬上撈一下，撈出來

接著發現負轉正的 dijkstra 可以處理 2 子題

所以做一做，燒了一下機，然後最終 debug 出來，也解掉了

剩下滿分解還沒想到，但是感覺好難

第一直覺是二分搜，但是我特別不會想題目

而且是比賽的一開始，我還是打算寫一些不用思考的題目

所以就跳題了

跑去看 pD

發現pD 好像是 yt 上有看到過的一題數學題

雖然我只有看那個影片的封面，但我很快就理解題目在幹嘛

然後因為我特別不會賽局，所以就跳題跑去寫 pE

發現 pE 根本就 cf 上的一題題目

很快就把他寫出來了

然後測範測測到第三筆測資發現燒機，答案不對

但我想不出為什麼錯，所以我就很生氣

傳 submission 上去，想說生氣的撈一下子題

然後就AC 了

幹，他範側三是錯的 什麼得真

然後我就跳去 pF

把第一個子題看出來是 靜態MST

把2、3子題想成 kruscal 重構樹

然後跳題跑去看 pG

發現 pG 是一題類似幾何的東西

所以我就跑回去寫 pF 子任務

簡單寫個MST，拿了第一個子任務

然後花了一個 30 min 刻 kruscal，才發現拔邊 + 建邊會讓樹的結構炸開

這時候我就破房了，好像 2 小時了，然後我分數才在 250 ~ 300 左右

整個大破防

然後我就很生氣的去寫 pG

子題一寫了個暴力最近點隊

子題二寫了位元枚舉

然後想 k = 2，想到一半就沒想了(方向好像是對的)

因為很急，感覺不能沒做事，所以我又跑去看 pH

然後一看，挖草，看不懂題目，然後又更急了

跑去看 pI 發現子題一跟二好像是科朵莉

然後就做了一下，然後傳傳傳發現 TLE 才發現複雜度估錯

但稍微調一下 code 也過了

然後盲猜子題三是啥怪資結 所以果斷跳題(阿結果是DP優化 哈哈)

翻了一下題本，靠，我沒題目可以做了

剩下的都是我覺得很難啃的東東

然後我就又破房了

時間好像只剩 2 小時

我根本不知道我在幹嘛

好像拿了 300 初而已

然後我就邊破房的思考邊想著這場打不好會被嗆 + 沒有大學可以讀

所以我就回去刻 pD，因為我想到一個很酷的 $O(N^4)$ 解法

阿這時候我已經有點放棄比賽了，因為發現其他東西我特別不會思考，感覺想了也沒用

所以就無情實作

然後燒機 1 個小時還是做不出來，超級生氣

然後我就跑去看了其他題目，又回去看了 pB，發現還是看不懂題目

回去看 pH，稍微理解一下題目後寫了一個樹直徑才發現他給的是圖

破防後寫了一個 $O(N(N+M))$ 的 BFS，然後撈了第一子題

就烙跑了

這時候就剩 30 min

我很失望的跳回去 pC

然後突然靈光一現，發現我二分搜的順序跟枚舉點的順序調換一下就可以解出來了

所以我就果斷把整份 code 刪掉，然後只花了 5 min 就做出來滿分解，我超強

然後轉回去 pF，突然發現我寫到一半的時間線段樹(某一個時間有刻+想的東西，但我忘記甚麼時候了)的 bug，修一下也拿到那個分數了

然後我就仔細觀察了一下我的解題狀況

只剩下 pD、pB 沒有撈到分數

所以我很生氣，跑回去debug pD，然後不知道為甚麼，突然就解出來了 ㄏㄏ

阿因為解出來 $O(N^4)$ 的解很開心

所以我就順便寫了一個特判 $p = 0$ 的解

這邊說一下，這個解只是把這個情況判掉，然後其他還是用 n <= 50 的處理

結果我傳上去子任務2挖，子任務 1 對了

然後我十分的矇，wtf，所以理論上是他子任務 2 卡的比子任務 1 還好

表揚全國賽出題者

然後剩最後十分鐘，我努力看了一下 pB

好像有點理解題目意思後，我翻了一下 code

突然發現中位數不是 sort 後的 size/2 的位置

而是要 $\frac{(\text{最初項} + \text{最後項})}{2}$

然後改了改就撈到 4 分

然後比賽就結束了

## 結語

這次全國賽應該是去玩的

真正要說的話，痛苦的應該是全國賽的前幾個禮拜

我都有在認真練題，雖然感覺沒用到，但很認真就是很認真

然後前一個禮拜也有認真調時差，所以這次稍微沒有出現賽中想睡覺的情況

(但是現在凌晨打心得，時差還沒調回來，我整個感覺快猝死 草)

全國賽中成為了全場除了一等一的子任務撈分高手

然後就沒有然後了

進了選訓感覺交大特選很穩

所以我會對清大說

> ~~曾經的我你看不起，如今的我你高攀不起~~ (沒)
